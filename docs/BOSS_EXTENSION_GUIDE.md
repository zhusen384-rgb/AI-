# Boss 浏览器扩展模式

## 目标

在同一个 Chrome 浏览器中，同时登录：

- 面试官系统
- Boss直聘

然后通过本地扩展，让面试官系统与当前已登录的 Boss 标签页协同工作，避免手动复制 Cookie，也避免独立自动化浏览器黑屏问题。

## 扩展目录

项目内扩展目录：

```text
browser-extension/boss-connector
```

## 安装步骤

1. 打开 Chrome
2. 访问 `chrome://extensions/`
3. 打开右上角 `开发者模式`
4. 点击 `加载已解压的扩展程序`
5. 选择项目目录中的：

```text
browser-extension/boss-connector
```

6. 安装完成后，刷新面试官系统页面

## 使用方式

1. 在同一个 Chrome 中登录 Boss直聘
2. 打开面试官系统 `自动打招呼 -> 平台账号`
3. 在“同浏览器 Boss 扩展模式”卡片中点击 `检测扩展`
4. 确认扩展已连接，并能看到 Boss 标签页
5. 选择目标 Boss 标签页
6. 点击 `读取当前 Boss 页面`
7. 如果能读到当前 Boss 页面信息，说明扩展协同链路已通

## 当前已支持

- 检测扩展是否已安装
- 列出当前浏览器中的 Boss 标签页
- 读取当前 Boss 页面状态
- 在 Boss 推荐牛人页执行：
  - 获取页面信息
  - 选择岗位
  - 点候选人卡片查看简历
  - 点击“打招呼”

## 注意

- 扩展模式依赖你手动登录的 Boss 标签页
- 如果新装扩展后页面无变化，先刷新面试官系统页面
- 如果 Boss 标签页没有被识别，确认标签页地址是 `https://www.zhipin.com/...`
