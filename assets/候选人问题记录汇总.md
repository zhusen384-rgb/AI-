# 候选人问题记录汇总

> **说明**：本文件用于汇总所有候选人在面试**第三阶段（公司介绍和问答）**提出的问题及 AI 的回答。
> 
> - 第一阶段（自我介绍）和第二阶段（核心问题提问）的内容**不记录**。
> - 所有候选人的问答统一汇总到本文件，不再单独创建文件。
> - 每条记录清晰标注对应的候选人信息，便于追溯。

## 记录格式

每条记录包含以下字段：
- `interviewId`: 面试ID
- `candidateName`: 候选人姓名
- `position`: 应聘岗位
- `timestamp`: 记录时间
- `question`: 候选人问题
- `answer`: AI回答
- `type`: 记录类型（candidate_question / interviewer_answer）
- `stage`: 面试阶段（固定为3）

## 使用说明

1. 面试进行到第三阶段时，系统会自动记录候选人的提问
2. 记录保存在 `assets/候选人问题记录汇总.json` 文件中
3. 可通过 API `/api/full-ai-interview/candidate-questions` 查询记录
